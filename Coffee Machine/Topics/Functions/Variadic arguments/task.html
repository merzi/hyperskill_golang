Variadic arguments

Emma has created a function divide that can divide a list of float (decimal) numbers. However, she forgot a very important detail to actually make her function able to take multiple arguments.

Can you help Emma fix her function so that it can take multiple float64 numbers as arguments?

Take notice that to solve this task, it is not necessary to change any code within the main function; the objective is to fix the parameter type of the divide function for it to be able to properly take multiple arguments of the float64 type!

Sample Output:

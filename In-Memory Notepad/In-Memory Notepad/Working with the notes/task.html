Description

Your notepad is already handy, but it's not that user-friendly. For example, you cannot delete an individual note; you need to delete all of them only to create the ones you really need. Let's improve this by adding two new commands – update and delete.

Objectives

Modify the solution for the previous stage by adding new commands – update and delete.

When the command is update:

  • The program should use a space (" ") as a separator to split the data into two pieces – the position (an integer) and the text (a string). Update the text of a note at the given position.

When the command is delete:

  • The program should interpret the data as a position, delete a note at the given position, and clear this position. Be careful to properly update the remaining positions: the position of a note should represent the order of the notes.

The program has to check the position and note before executing the update or delete commands:

  • If there is no note at the given position, print [Error] There is nothing to update\n or [Error] There is nothing to delete\n;
  • If the program cannot convert the given position-string to the position(an integer), print [Error] Invalid position: %s\n, where %s is the string;
  • If the program cannot find the position-string, print [Error] Missing position argument\n;
  • If the program cannot find the note-string, print [Error] Missing note argument\n;
  • If the position is out-of-boundaries, print [Error] Position %d is out of the boundaries [1, %d]\n, where the second %d is the internal storage size;
  • If the operation was successful, print [OK] The note at position %d was successfully updated\n or [OK] The note at position %d was successfully deleted\n.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1: program execution; one of the update and delete command is followed by a few spaces

Enter the maximum number of notes: > 3

Enter a command and data: > list
[Info] Notepad is empty

Enter a command and data: > create This is my first record!
[OK] The note was successfully created

Enter a command and data: > list
[Info] 1: This is my first record!

Enter a command and data: > update 1
[Error] Missing note argument

Enter a command and data: > update one Updated first note!
[Error] Invalid position: one

Enter a command and data: > update
[Error] Missing position argument

Enter a command and data: > update 1 Updated first note!
[OK] The note at position 1 was successfully updated

Enter a command and data: > list
[Info] 1: Updated first note!

Enter a command and data: > delete one
[Error] Invalid position: one

Enter a command and data: > delete
[Error] Missing position argument

Enter a command and data: > delete 1
[OK] The note at position 1 was successfully deleted

Enter a command and data: > list
[Info] Notepad is empty

Enter a command and data: > exit
[Info] Bye!
